# SiBibit Desktop Dashboard

Aplikasi desktop dummy/mock untuk Raspberry Pi berbasis Python + PyQt5.

## Struktur

```text
main.py
backend/
  mock_data.py
pages/
  dashboard.py
  history.py
  login.py
  settings.py
  statistics.py
  splash.py
styles/
  theme.qss
widgets/
  charts.py
  common.py
  sidebar.py
```

## Menjalankan

```bash
pip install -r requirements.txt
python main.py
```

Semua data, sensor, kamera, dan AI detection masih dummy/simulasi.
